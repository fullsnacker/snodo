## Iniciando el proyecto

### Instalacion

1. Clona el repositorio
   ```sh
   git clone https://github.com/fullsnacker/snodo.git
   ```
2. Instala los paquetes YARN
   ```sh
   yarn
   ```
3. Lanza el proyecto con:
   ```sh
   yarn dev
   ```
